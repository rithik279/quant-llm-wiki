SECTION 2: BREAKOUT STRATEGY FOUNDATION

Initial system:

* Breakout screener identifies stocks making new highs
* Outputs different tickers daily (rolling signals)

Proposed baseline test:

* Entry: buy on signal day (close or next open)
* Exit: fixed holding period (~40 trading days / 2 months)

Data structure:

* Each (date, ticker) = one independent trade
* Dataset includes:

  * date
  * ticker
  * entry price
  * exit price
  * return

Evaluation approach:

1. Signal-level evaluation:

   * mean return
   * median return
   * win rate
   * return distribution
   * tail behavior

2. Portfolio-level simulation:

   * max positions constraint (e.g., 10)
   * equal weighting
   * rolling entries/exits
   * daily rebalancing
   * track:

     * portfolio value
     * drawdown
     * Sharpe

3. Time segmentation:

   * evaluate across:

     * bull markets
     * bear markets
     * sideways regimes

4. Time decay analysis:

   * returns at:

     * 5 days
     * 10 days
     * 20 days
     * 40 days

Observations:

* signals overlap (non-independent)
* distribution more important than mean
* momentum expected to have fat tails

SECTION 3: POST-SCREENER SYSTEM DESIGN

Upgrade from:

* “list of breakouts”

To:

* “ranked + filtered breakout candidates”

Four scoring dimensions:

1. Trend Strength

   * 20D return
   * 60D return
   * distance from moving averages
   * price structure

2. Relative Strength

   * vs QQQ
   * vs sector
   * leadership detection

3. Confirmation

   * volume spike
   * follow-through
   * consolidation quality
   * rejection behavior

4. Growth / Fundamentals

   * EPS revisions
   * revenue growth
   * earnings surprises
   * guidance

Example weighting:

* Trend: 35%
* Relative Strength: 30%
* Confirmation: 20%
* Growth: 15%

SECTION 5: REGIME CLASSIFICATION

Three-state model:

* Trend (risk-on)
* Chop (mean-reversion)
* Risk-off (downtrend)

Initial approach: rule-based

Features:

* index vs 50DMA / 200DMA
* breadth (% above MA)
* volatility
* breakout performance

Later approach: Hidden Markov Model (HMM)

HMM inputs:

* index returns
* volatility
* breadth
* momentum factor returns
* breakout strategy returns

Output:

* latent states mapped to regimes

Alternative approach:

* classify regimes directly from strategy performance
  (“when breakouts work vs fail”)

Usage:

* adjust:

  * position sizing
  * trade frequency
  * entry strictness
